# 画面設計書：AI チャット英語学習システム

## 1. 概要

本ドキュメントは、AI チャット英語学習システムの全画面設計を定義する。
対応環境は PC ブラウザ（Chrome / Safari / Edge 最新版）およびスマートフォンブラウザ（iOS / Android）。
全画面レスポンシブデザインに対応する。

---

## 3. 共通仕様

### 3.1. ヘッダー（認証済み画面共通）

PC ではサイドバーがナビゲーションを担うため、ヘッダーにはアプリ名のみを表示する。
モバイルではヘッダーにアプリ名を表示し、ナビゲーションはボトムナビゲーションが担う。

| 要素 | PC | モバイル |
|---|---|---|
| アプリ名 | 表示（コンテンツエリア上部）。クリックで S-01 へ遷移 | 表示（ヘッダー中央）。クリックで S-01 へ遷移 |
| ブックマークアイコン | **非表示**（サイドバーに移管） | **非表示**（ボトムナビゲーションに移管） |
| 設定アイコン | **非表示**（サイドバーに移管） | **非表示**（ボトムナビゲーションに移管） |

### 3.2. トースト通知（共通）

操作の完了・エラーをトースト通知で表示する。

| 種別 | 表示内容例 | 表示時間 |
|---|---|---|
| 成功 | 「ブックマークに保存しました」 | 3 秒 |
| エラー | 「エラーが発生しました。再度お試しください。」 | 5 秒 |

### 3.3. ローディング状態

API 呼び出し中はボタンを非活性化し、スピナーを表示する。

### 3.4. レスポンシブ対応方針

| ブレークポイント | 幅 | ナビゲーション | レイアウト |
|---|---|---|---|
| モバイル | 〜 767px | ボトムナビゲーション（画面下部固定） | 1カラム、全幅 |
| PC | 768px 〜 | 左固定サイドバー（幅 220px） | サイドバー + コンテンツエリア |

---

### 3.5. サイドバー / ボトムナビゲーション（認証済み画面共通コンポーネント）

認証済みの全画面（S-01 / S-02 / S-07）に共通して表示するナビゲーションコンポーネント。

#### PC：左固定サイドバー

```
┌────────────────────┐
│  🗣️  English Chat  │  ← アプリロゴ・名称（クリックで /chat へ）
├────────────────────┤
│                    │
│  💬  チャット       │  ← アクティブ時: 背景色変更・左ボーダー強調
│                    │
│  🔖  ブックマーク   │
│                    │
├────────────────────┤  ← スペーサー（残り高さを埋める）
│                    │
│  👤  表示名         │  ← ユーザー情報（アバター + 表示名）
│                    │
│  ⚙️  設定          │
│                    │
│  ↩️  ログアウト     │
└────────────────────┘
```

| 要素 | 説明 |
|---|---|
| アプリロゴ・名称 | サイドバー最上部。クリックで `/chat`（S-01）へ遷移 |
| 💬 チャット | `/chat`（S-01）へ遷移。現在の画面と一致する場合はアクティブ状態で強調表示 |
| 🔖 ブックマーク | `/bookmarks`（S-02）へ遷移。アクティブ状態切り替えあり |
| ユーザー情報 | アバターアイコン + `display_name` を表示（Supabase `profiles` から取得） |
| ⚙️ 設定 | `/settings`（S-07）へ遷移。アクティブ状態切り替えあり |
| ↩️ ログアウト | `signOut` Server Action を呼び出し → `/login` へリダイレクト |

**アクティブ状態の視覚的表現:**
- 現在表示中の画面に対応するメニュー項目は背景色を変更（例：薄いアクセントカラー）
- 左端にカラーボーダーを表示（例：2〜4px の太さ）

---

#### モバイル：ボトムナビゲーション

```
┌──────────────────────────────────────────────┐
│                                              │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  │
│  │    💬    │  │    🔖    │  │    ⚙️    │  │
│  │ チャット │  │ブックマーク│  │   設定   │  │
│  └──────────┘  └──────────┘  └──────────┘  │
│                                              │
└──────────────────────────────────────────────┘
```

| 要素 | 説明 |
|---|---|
| 💬 チャット | `/chat`（S-01）へ遷移。アクティブ時はアイコン・テキストを強調色で表示 |
| 🔖 ブックマーク | `/bookmarks`（S-02）へ遷移。アクティブ状態切り替えあり |
| ⚙️ 設定 | `/settings`（S-07）へ遷移。アクティブ状態切り替えあり |

**注意:** チャット画面（S-01）ではテキスト入力フィールドがフッターに固定されるため、ボトムナビゲーションはその**上**に配置する。キーボード表示時はボトムナビゲーションを非表示にすることを推奨する。

---

#### Next.js 実装上の配置

サイドバー / ボトムナビゲーションは認証済みレイアウトの共通コンポーネントとして `app/(auth)/layout.tsx` に配置する。

```
app/
├── (auth)/                   # 認証済みレイアウトグループ
│   ├── layout.tsx            # サイドバー / ボトムナビゲーションを含む共通レイアウト
│   ├── chat/
│   │   └── page.tsx          # S-01
│   ├── bookmarks/
│   │   └── page.tsx          # S-02
│   └── settings/
│       └── page.tsx          # S-07
└── (public)/                 # 未認証レイアウトグループ（サイドバーなし）
    ├── login/page.tsx        # S-04
    ├── signup/page.tsx       # S-05
    └── reset-password/page.tsx # S-06
```

### 4.2. 要素定義

#### ヘッダー

| 要素 | PC | モバイル | 説明 |
|---|---|---|---|
| アプリ名 | 表示 | 表示（中央） | クリックで `/chat` へ遷移 |
| ブックマークアイコン | 非表示 | 非表示 | サイドバー / ボトムナビゲーションに移管 |
| 設定アイコン | 非表示 | 非表示 | サイドバー / ボトムナビゲーションに移管 |

#### チャットログ

| 要素 | 説明 |
|---|---|
| AI メッセージ | 左寄せ、吹き出し形式 |
| ユーザーメッセージ | 右寄せ、吹き出し形式 |
| 提案メッセージ吹き出し | AI メッセージの後に最大 3 件表示。各吹き出しは独立 |

#### 提案メッセージ吹き出し（1件あたり）

| 要素 | 説明 |
|---|---|
| 英語例文 | 大きめのフォントで表示 |
| 日本語訳 | 英語例文の下に小さめのフォントで表示 |
| 音声アイコン（🔊） | クリックで当該英文の音声を再生（Web Speech API） |
| ブックマークアイコン（🔖） | 未登録時：白抜き / 登録済み：塗りつぶし。クリックでブックマーク登録/解除のトグル |

#### フッター

| 要素 | 説明 |
|---|---|
| テキスト入力フィールド | プレースホルダー「メッセージを入力...」。最大 1000 文字 |
| 送信ボタン（➤） | クリックまたは Enter キーで送信。AI 生成中は非活性 |

### 4.3. 状態定義

| 状態 | 表示内容 |
|---|---|
| 初期表示 | AI の初回メッセージ（「こんにちは！今日はどんな英語を学びたいですか？」）が表示される |
| AI 生成中 | チャットログ末尾にローディングアニメーション（...）を表示。送信ボタンを非活性化 |
| ブックマーク済み | ブックマークアイコンが塗りつぶし状態になる |
| 音声再生中 | 音声アイコンが再生中状態に変化（例：波形アニメーション） |

### 4.4. インタラクション

| 操作 | 結果 |
|---|---|
| メッセージ送信 | ユーザーメッセージを右寄せで表示 → AI 生成中アニメーション → 提案メッセージ 3 件を順次表示 |
| 音声アイコンクリック | 英語例文の音声を再生（2 秒以内に開始） |
| ブックマークアイコンクリック（未登録） | ブックマーク登録 → アイコンが塗りつぶしに変化 → 「ブックマークに保存しました」トースト |
| ブックマークアイコンクリック（登録済み） | ブックマーク解除 → アイコンが白抜きに変化 |
| チャットログのスクロール | 上方向にスクロールして過去のメッセージを閲覧可能 |
